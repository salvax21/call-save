INSTRUCCIONES PARA SUBIR A GITHUB PAGES

1. Crea una cuenta en https://github.com si aún no tienes una.
2. Crea un nuevo repositorio con cualquier nombre (por ejemplo: callsave).
3. Sube el archivo `index.html` a tu nuevo repositorio.
4. Ve a la configuración (Settings) del repositorio.
5. En la sección "Pages", selecciona la rama `main` y la carpeta raíz (/root).
6. Guarda y espera unos segundos. Tu sitio estará disponible en: https://tuusuario.github.io/nombre-repositorio

¡Listo! Tu página ya estará pública.
